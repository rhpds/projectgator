# Screenshots

This directory contains screenshots of Projectgator in action.

## Screenshots Available

- `home.png` - Home page with feature overview
- `projects-empty.png` - Projects page empty state
- `projects-list.png` - Projects page with project cards
- `tasks-empty.png` - Tasks page empty state
- `tasks-list.png` - Tasks page with task cards
- `dark-mode.png` - Dark mode interface
- `create-project-modal.png` - New project creation modal
- `create-task-modal.png` - New task creation modal

## Capturing Screenshots

To update screenshots:

1. Start the application locally
2. Navigate to the desired page
3. Use browser developer tools or screenshot tools
4. Save as PNG at 1920x1080 resolution
5. Optimize images before committing
